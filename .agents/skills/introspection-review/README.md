# introspection-review

## Overview

Manual bounded classification workflow for persisted Agent Introspection candidates.

## When to use it

- A user explicitly requests classification of persisted candidates.
- Current capability proof is required before classification begins.

## Example prompts

- “Classify the current Agent Introspection candidates.”
- “Run a manual candidate classification and report accepted results.”
