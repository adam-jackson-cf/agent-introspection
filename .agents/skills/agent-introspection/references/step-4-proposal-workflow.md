# Proposal workflow

## Objective

Persist complete intervention proposals while preventing application.

## Required actions

1. Require an actionable finding and validated proposal review.
2. Apply the deterministic intervention hierarchy.
3. Persist every mandatory proposal field and append-only event.
4. Display the proposal without changing its target.

## Done when

- The proposal is persisted in pending state with complete evidence.
- No target repository has been mutated.
